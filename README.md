# Ecodistrict

Ecodistrict — Қазақстан қалаларының аудандарының экологиялық жағдайын талдайтын интерактивті веб-сайт. Пайдаланушылар қала мен ауданды таңдап, экологиялық рейтингті және деректерді көре алады.

## Ерекшеліктер

- Аудандардың экологиялық көрсеткіштерін картада көрсету.
- Қалалар мен аудандар бойынша сүзу.
- Деректерді визуализациялау үшін интерактивті элементтер.
- Статикалық сайт, GitHub Pages арқылы жарияланған.

## Технологиялар

- HTML5, CSS3  
- JavaScript  
- Leaflet.js (карталар үшін)  
- GitHub Pages (жариялау үшін)
